The World Wide Web and HTML

HTML documents make up the world wide web. A person receives HTML documents via HTTP request sent to and from a web browser. Here is a link to a video for further explination.

HTML stands for Hyper Text Markup Language. HTML documents form the majority of the content on the web. HTML documents contain text content which describes "what you see" and markup which describes "how it looks". For another overview here is a good video.


Elements,Tags, and Attributes

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags. Example below:

Tags: "b" tag (bold) or "em" tag (italics or emphais). Tags are made up of an open tag and a closed tag with some sort of contents between them. The contents that falls between the open and close tags is what the tags are being applied to.

Attributes: "a" (anchor) attribute is for creating links. This attribute is made up of the attribute name (href), value (web address "www.google.com"), and discripter word that will be highlighted indicating a link is present.

Image: "img" (image) tags are VOID tags, which means they don't have content. They are made up of src (source or location of saved image) and alt (alternate text to be displayed when image fails to load.

White Space

White space is defined as all "blank" spaces, tabs, or returns. In HTML all white space is seen as a single space. A return line tag has to be used display multiple lines. Examples below:

Return Line: "br" ( break) or "p" (paragraph). break tags are void tags, where paragraph tags are not void. This means break tags are only a signle tag (no open and close tag) and paragraph tags have and open tag, content, and a close tag.

Inline/Block: inline tags are inline with text(br). Block tags have contents and place a block or box around the contents that has both hight and width.


Container Elements

Container elements are elements with tags that contain a section of text inside HTML. Examples are below:

SPAN: is an inline tag

DIV: is a block tag
